### How to run the application
- `javac App.java` and then `java App`
- OR
- `java -jar optimizer-1.0.jar <input_file_name.json>`
- make sure the input json file is in the same directory as the jar file


### Pre-requisites
- Java 8
- Maven 3
- Mac or Linux OS
